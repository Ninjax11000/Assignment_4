1. What is DOM and what is the purpose of DOM?

    Ans:DOM means Document Object Model.It represents the web page so that it can be manipulated using programming language.It represents the web page as nodes and objects.

  2. How will you select HTML elements using DOM? Name the DOM methods.

    Ans:We can select elements in a DOM, using DOM methods.The methods are-

    • getElementsByTagName()
    • getElementsByClassName()
    • getElementById()
    • querySelector()
    • querySelectorAll()

  3. What is event bubble?

    Ans:Event Bubble is a method for propagation in HTML DOM. It happens when an event happens on an element, it first runs the handlers on it. Then on its parent, then on the current nodes parent such that Event handler is executed to the top most parent.

  4. What is a callback function and why will you use it?

    Ans:When a function is passed to a function as an argument, which is then called inside the outer function to do specific task, that passed function is called callback function.I will use callback function when i want a function to be executed after the execution of another function
